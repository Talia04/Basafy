# Basafy (Expo + React Native)

Fresh Expo scaffold ready for Basafy mobile app work.

## Quick start
```bash
npm install
npm run start          # opens Expo Dev Tools; press i/a/w for iOS/Android/Web
```
Shortcuts:
- `npm run ios`
- `npm run android`
- `npm run web`

## Project layout
- `App.tsx` — starter placeholder screen.
- `app.json` — Expo config (icons/splash placeholders in `assets/`).
- `babel.config.js`, `tsconfig.json`, `package.json` — tooling + deps (Expo SDK 51).
